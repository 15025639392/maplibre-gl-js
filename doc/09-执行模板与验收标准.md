# 09 执行模板与验收标准

## 1. 设计提案模板（RFC）

```md
# RFC: <标题>

## 背景
- 现状与问题
- 目标与边界

## 方案
- 架构图
- 接口定义
- 兼容性影响

## 替代方案
- 方案 A/B 对比

## 风险与回滚
- 风险清单
- 回滚策略

## 验收标准
- 功能
- 性能
- 质量门禁
```

## 2. 迭代周报模板


## 5. 里程碑验收清单（DoD）

- [ ] 功能需求完成并通过评审
- [ ] 质量门禁（lint/typecheck/test）通过
- [ ] 性能指标达标或有解释
- [ ] 兼容性验证完成
- [ ] 文档与变更日志更新
- [ ] 回滚方案已验证

## 6. 发布前检查清单

- [ ] 变更说明与升级指南已发布
- [ ] 灰度策略已配置
- [ ] 监控告警阈值已确认
- [ ] 版本 Tag 与制品校验通过
- [ ] 回滚脚本可执行
