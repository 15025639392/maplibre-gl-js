# 变更包模板说明

## 使用方式

1. 复制本目录并重命名：`YYYY-MM-DD-type-slug`
2. 填写 `00` 到 `05` 六个文档
3. 在 `doc/changes/INDEX.md` 增加索引记录
4. 完成后根据 `05-回写主文档清单.md` 回写主文档

## 模板文件

- `00-变更摘要.md`
- `01-方案设计.md`
- `02-任务拆解.md`
- `03-测试与验收.md`
- `04-发布与回滚.md`
- `05-回写主文档清单.md`

## 最低完成标准

- 00：目标、范围、影响面明确
- 01：方案与兼容性结论明确
- 02：任务与负责人明确
- 03：测试记录与验收结论明确
- 04：发布、回滚与复盘信息明确
- 05：回写结论明确（或豁免原因明确）
