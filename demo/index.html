MapLibre Local Demo Hub

This folder is a dedicated multi-page demo workspace for local development.

Run npm run start-demo, then open /demo/.

Tip: All map demos load dist/maplibre-gl-dev.js and dist/maplibre-gl.css, so source changes appear automatically when watch tasks are running.
